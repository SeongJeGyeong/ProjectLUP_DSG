# 소개
Unity6를 기반으로 하나의 프로젝트 내에 각 팀 별로 미니 게임을 제작한 프로젝트입니다.<br/>
해당 프로젝트 내에서 턴제 RPG 게임인 DSG의 개발을 담당했습니다.

+ 개발 환경 : Unity 6000.0.62f1, Git(Fork)
+ 플랫폼 : PC
+ 개발 인원 : 3명
